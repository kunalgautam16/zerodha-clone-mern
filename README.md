# Zerodha Clone MERN

A full-stack Zerodha-inspired trading platform built using the MERN stack featuring authentication, role-based access, ticketing system, and interactive trading dashboard.

---

## Features

- User Signup & Login Authentication
- JWT-based Secure Authentication
- Protected Dashboard Routes
- Holdings & Positions Dashboard
- Support Ticket Creation System
- MongoDB Database Integration
- Responsive UI inspired by Zerodha
- Logout Functionality
- Role-Based Access Foundation
- MERN Stack Architecture

---
